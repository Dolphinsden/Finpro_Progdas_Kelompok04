# Finpro_Progdas_Kelompok04

Kelas : 01
Kelompok : 04
SDG : 15 (Life on Land)

Judul : Program Analisis Kondisi Ekosistem Darat dan Perubahannya Setiap Tahun 

Deskripsi : Program ini merupakan sistem simulasi yang memungkinkan pengguna untuk menganalisis kondisi ekosistem darat secara berkelanjutan. Program dapat menyimpan dan mengolah data seperti luas hutan, tingkat degradasi lahan, dan jumlah keanekaragaman hayati. Program ini mampu mengklasifikasikan kondisi lingkungan ke dalam beberapa kategori (baik, sedang, kritis) serta melakukan analisis sederhana terhadap perubahan data dari waktu ke waktu. Program ini bertujuan untuk meningkatkan kesadaran bangsa terhadap perubahan ekosistem.
